# NGO-Volunteer-Dashboard

An interactive Power BI dashboard designed to manage 1,000 volunteers across Maharashtra and Gujarat. This tool helps NGOs match the right people with the right causes at the right time.

Key Features Volunteer Overview: Tracks 1,000 active members with filters for Gender and ID. Skill Mapping: Identifies top expertise in Teaching and Event Management. Training Tracker: Shows that 45.9% of volunteers are fully trained, highlighting a need for more onboarding. Availability: Monitors schedules (Full-time, Weekends, Evenings) to ensure event coverage. Geo-Intelligence: A live map pinpointing volunteers in cities like Mumbai, Pune, and Nagpur. Retention Insights: Tracks satisfaction vs. experience to prevent volunteer burnout.

Tech Stack Tool: Power BI Desktop Data Model: Star Schema (Volunteer, District, and Skill tables) Visuals: Custom "Humanitarian" theme with interactive slicers and maps.

Top Findings Training Gap: Over 50% of volunteers still need training. Resource Hubs: Kolhapur and Nagpur have the highest volunteer density. Skill Needs: While Teaching is high, we need to recruit more for Logistics and Medical Support.
