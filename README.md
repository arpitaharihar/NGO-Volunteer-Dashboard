1. Project Title / Headline
NGO Volunteer Analytics Dashboard using Microsoft Power BI

This project focuses on analyzing volunteer data of NGOs using interactive data visualization. The dashboard provides insights into volunteer skills, availability, training status, and geographical distribution to help NGOs manage volunteers effectively.

2. Short Description / Purpose

The NGO Volunteer Dashboard is an interactive data visualization project created using Microsoft Power BI to analyze and monitor volunteer information.
The main purpose of this dashboard is to help NGO managers and administrators understand volunteer participation and engagement. By analyzing the data, organizations can easily identify:

* Which skills are most available among volunteers
* Which districts have more volunteer participation
* Volunteer availability schedules
* Training completion status
* Volunteer satisfaction levels
* The relationship between experience and satisfaction
This dashboard helps NGOs make data-driven decisions for volunteer management, better task assignment, and improving volunteer programs.


3. Tech Stack

The following technologies and tools were used to build this project:
a. Data Visualization Tool :Microsoft Power BI
Used for:
* Creating interactive dashboards
* Data visualization
* Building charts and reports
* Applying filters and slicers

b. Data Processing
Data preprocessing techniques were used to:
* Clean missing values
* Organize the dataset
* Structure the data for analysis

c. Dataset Handling
The dataset was stored and processed in Excel / CSV format, which was then imported into Power BI.

d. Data Source
The dashboard is built using an NGO volunteer dataset that contains information about volunteers working in different districts and roles.

The dataset includes the following attributes:

| Column              | Description                                                 |
| ------------------- | ----------------------------------------------------------- |
| Volunteer_ID        | Unique ID for each volunteer                                |
| Volunteer_Name      | Name of the volunteer                                       |
| Age                 | Age of the volunteer                                        |
| Gender              | Male / Female                                               |
| District            | Location of the volunteer                                   |
| Primary_Skill       | Main skill such as Teaching, Medical Assistance, Counseling |
| Secondary_Skill     | Additional skills                                           |
| Availability        | Time availability (Full-time, Weekends, Evenings etc.)      |
| Training_Completed  | Indicates if volunteer has completed training               |
| Experience_Years    | Number of years volunteering                                |
| NGO_Name            | Organization where volunteer works                          |
| Satisfaction_Rating | Rating of volunteer experience                              |

This dataset allows organizations to analyze volunteer engagement patterns.

4. Features / Highlights of the Dashboard

The dashboard contains multiple visualizations that provide different insights.
i.Volunteer Distribution Analysis
Shows the number of volunteers across different districts and skill areas.

ii.Volunteer Availability Insights
Visualizes availability types like Full-time, Evenings, Flexible, Weekdays, and Weekends.

iii.Training Completion Tracking
Displays how many volunteers have completed training vs not completed.

iv. Skills & Category Analysis
Identifies volunteer skills across sectors like Child Care, Education, Healthcare, Environment, and Animal Welfare.

v.Experience vs Satisfaction Analysis
Line chart showing how volunteer experience impacts satisfaction levels across NGOs.

vi.Interactive Filters
Filters for Gender and Volunteer Name for deeper analysis.

vii. Geographical Visualization
Map visualization showing district-wise volunteer availability.

viii.Volunteer Database View
Tabular section displaying detailed volunteer information.
